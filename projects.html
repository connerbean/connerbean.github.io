MinVasion

  • A 2D Game that has users click-and-drag to avoid character-seeking minions


  • This game was created in September 2017. Written in C++, the game starts with a character, Gru, which users have to move around obstacles to avoid minions chasing him. Each obstacle he leads minions into are worth individual points. Players attempt to stay alive for as long as possible while also getting the highest score.

Super Mar-IO

  • A neural network designed with the intention of beating Super Mario on Continuous Play


  • This project was worked on at SpartaHack 2017 with Alex Lambert. I developed the fitness function in C# and Lua, which was responsible for determining how close the current generation was to victory. The program runs on BizHawk Emulator, and runs through continuous iterations of Mario's with the goal of defeating the game with the setting 'Continuous Play' on.

Technologies / Skills: C#, Lua, Statistics, Analysis
